Pong(Понг), выпущенный Atari в 1972 году, стал первой коммерчески успешной видеоигрой, созданной Алланом Алькорном по заданию Нолана Бушнелла.
Это простая игра в виртуальный теннис с двумя ракетками и мячом.
Основные правила игры Pong:
1. Игроки (или игрок против компьютера) управляют прямоугольными ракетками по левую и правую стороны экрана, перемещая их вертикально.
2. Нужно отбивать мяч ракеткой так, чтобы он не вышел за пределы "игрового поля"
3. Если игрок пропускает мяч, его соперник получает одно очко. Игра продолжается до достижения определенного счета.
4. С течением времени скорость движения мяча постепенно увеличивается, усложняя игровой процесс, а так же ракетки не могут доходить до самого верхнего края экрана, что тоже усложняет игру.
